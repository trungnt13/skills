# trungnt13-skills

All-in-one bundle of trungnt13's skills and agent bundles for GitHub Copilot CLI and Claude Code.

This plugin bundles the full collection:

- skills in `skills/`
- agent bundles in `agents/`

## Included Agents

- `deepdive` — first-principles research and architecture tracing
- `boris` — plan-first coding workflow adapted from Boris Cherny's public Claude Code guidance
- `consensus` — multi-model answer synthesis
- `git-commit-push` — staging, conventional commits, and safe push workflow

## Included Skills

- `vscode`
- `obsidian-markdown`
- `obsidian-bases`
- `json-canvas`
- `obsidian-cli`
- `defuddle`
